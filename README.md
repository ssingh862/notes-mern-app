# notes-mern-app
# Notes MERN App 📝

A full-stack Notes Application built with the **MERN** stack (MongoDB, Express.js, React, Node.js). This app allows users to sign up, log in, create, update, and delete personal notes.

---


---

## 📦 Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Other Tools:** Postman (for API testing), Git, GitHub

---
